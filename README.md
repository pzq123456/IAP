# 智慧景区平台 (Intelligent Attraction Platform)


## 选题背景

## 功能点12

### 非地图功能

#### 景点介绍、游客评论、食宿信息

#### 天气（以小组件形式展示）

#### 景区票务管理：预定、订单查询、退票

### 需要地图展示与交互的功能

#### 部分建筑物及景观三维展示

#### 景区地图

#### 景点位置

#### 路线规划（最短路径查询）

#### 客流情况（热力图）

#### 实时公交

#### 停车场、厕所及其他设施位置查询（缓冲区查询）

#### 三维景点展示

## 项目架构
- 前端：
  - 前端框架：Vue.js
  - 前端UI框架：Element Plus
  - 地图：百度地图API
  - 三维展示：three.js
  - 数据可视化：Echarts、Chart.js
  - 表格渲染：Ag-Grid
- 后端：
  - 后端框架：Flask + SQLAlchemy
  - 数据库：SQLite
